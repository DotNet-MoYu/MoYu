## 🍭 介绍

一个应用程序脚手架，您可以将它集成到任何 .NET/C# 应用程序中。

## 🎈 名字的由来

`MoYu` 顾名思义 `摸鱼` 的意思，表示用了这个框架就可以有大把时间摸鱼,有更多的时间学习新技术。

## 🎉 开源地址

- GitHub：[https://github.com/DotNet-MoYu/MoYu](https://github.com/DotNet-MoYu/MoYu)

## 安装

```powershell
dotnet add package MoYu
```
